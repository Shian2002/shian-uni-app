## 已发现并修正的问题

- 旧候选包 `artifacts/store-screenshots/2026-06-15T-profile-delete-focused-r4/` 中的 `03-agent-workbench-mobile-390-时安-agent-工作台.png` 实际显示的是登录弹窗，不应再作为 Agent 工作台截图。
- 截图脚本曾用页面全文判断关键词，登录弹窗遮挡时也可能误判通过。现已改为检查可见元素文字，并在 Agent、积分、账号注销截图前确认没有可见登录弹窗。
- 积分页截图曾只停留在顶部，现已滚动到能看到会员方案和短期问题积分说明的位置。
- 截图脚本曾在 Playwright 全局上下文里直接调用 `uni.setStorageSync`，但 H5 打包后的 `uni` 存储 API 并不是稳定浏览器全局对象，导致个人中心截图没有稳定登录态。现已改为先写 `localStorage`、验证 `/api/me` 真实会话，再复用 `__xcRenderTabPath` 渲染 tab 页面。
