# 塔罗牌图资源说明

本项目塔罗牌面使用经典 Rider-Waite-Smith 牌图作为本地静态资源，文件位于 `src/static/tarot/rws/`。

资源来源：

- GitHub: `metabismuth/tarot-json`
- 牌图目录：`https://github.com/metabismuth/tarot-json/tree/master/cards`
- 数据说明：`https://github.com/metabismuth/tarot-json/blob/master/tarot-images.json`

该仓库说明 Rider-Waite-Smith deck 在美国为 public domain，项目代码以 MIT License 发布。上线使用时保留本说明，便于后续核验来源与授权状态。
