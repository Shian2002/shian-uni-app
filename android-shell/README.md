# Android 调试壳

这个目录用于在 HBuilderX/DCloud 发行权限恢复前，先产出一个可安装 Android 调试 APK，方便真机验证登录、积分和时安 agent 线上功能。

- 包名：`com.shian.xuanctai`
- 版本：`1.0.0` / `100`
- 启动入口：`https://shianjieyouwu.com/#/?app=1`
- 图标来源：`src/static/app-icons`
- 产物用途：内部调试、真机验收、release inbox 收件证据

它不是商店正式包；正式上架仍优先走 HBuilderX/DCloud、Google Play AAB 或各国内渠道签名包。
